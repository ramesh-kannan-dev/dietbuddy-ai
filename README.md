# DietBuddy – AI Nutrition Planner

## 🚀 Overview
DietBuddy is a full-stack AI-powered application that generates personalized diet plans based on user inputs such as age, weight, and fitness goals.

## 🔥 Features
- Personalized 7-day meal plans
- Calorie, protein, and fat tracking
- AI chatbot for diet recommendations
- PDF export functionality

## 🛠 Tech Stack
- Backend: FastAPI (Python)
- Frontend: Next.js, React
- Database: SQLite, SQLAlchemy
- AI: Google Generative AI API

## ⚙️ How to Run

### Backend
pip install -r requirements.txt  
uvicorn main:app --reload  

### Frontend
npm install  
npm run dev
